# Discourse Konami

This gem adds the Konami Code to Discourse.

When invoked, magic happens.

## Installation

Add this line to your Discourse's Gemfile:

    gem 'discourse_konami', :git => 'https://github.com/cfstras/discourse-konami'

And then execute:

    $ bundle install

## Usage
